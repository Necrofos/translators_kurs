# Лабораторная работа №6  

### Тема
Разработка синтаксического анализатора для выражений из операции сложения и операций сравнения `<`, `>`, `=`, `!=` над целыми числами с использованием **ANTLR4** и построением **дерева разбора**.

### Язык реализации
C++

### Используемые технологии
- ANTLR4
- C++
- CMake

---

## Поддерживаемые конструкции

Анализатор поддерживает:

- целые числа (`INT`)
- операцию сложения `+`
- операции сравнения:
  - `<`
  - `>`
  - `=`
  - `!=`

Примеры корректных выражений:

```txt
2
2+3
1+2+3
2<3
5>1+1
7=3+4
8!=2+5
2+3<4
10+5>7+1
```
